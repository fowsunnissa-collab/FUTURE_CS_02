# FUTURE_CS_02
# Security Alert Monitoring & Incident Response Using Splunk Enterprise

## Overview
This project demonstrates the use of Splunk Enterprise as a SIEM tool to monitor security events, visualize log data, identify suspicious activity, and document incident response actions.

## Tool Used
- Splunk Enterprise (Free Trial)

## Log Sources
- Security and DNS-related logs

## Key Features
- Log ingestion and verification
- SPL-based security analysis
- Visualization using charts
- Centralized security dashboard
- Incident classification and response planning

## Visualizations
- Security Events Over Time
- Event Distribution by Log Type
- Top Hosts by Event Volume
- Top Event Sources

## Outcome
Successfully implemented a SIEM-based security monitoring solution using Splunk Enterprise.
